# logic_lecturenoteai
#  (AI 기반 강의 노트 자동 생성 및 오타 교정 파이프라인)

오픈소스 멀티모달 AI를 활용하여 대학교 강의 영상 등으로부터 슬라이드별 핵심 요약 노트를 자동 생성하고, 음성 인식 오타를 화면 정보를 통해 교차 검증하는 시스템입니다.

---

##  핵심 기술 스택
* **Language:** Python
* **Computer Vision:** OpenCV (Frame Differencing 기반 화면 전환 감지)
* **Speech-to-Text:** OpenAI Whisper (Hugging Face)
* **Vision-Language Model:** Qwen-VL (Qwen2.5-VL-7B-Instruct or Qwen3.5-9B(4bit 양자화))

##  주요 파이프라인 프로세스
1. **오디오/비디오 분리:** 업로드된 영상에서  오디오 추출 후 **Whisper** 구동 (타임스탬프 텍스트 추출).
2. **화면 감지 엔진:** **OpenCV**를 활용해 PPT 슬라이드가 넘어가는 프레임 전환 시점을 포착하여 자동 캡처 및 타임라인 기록.
3. **시간대별 데이터 매칭:** 캡처된 이미지의 시간축과 Whisper 대사의 시간축을 통합하여 세트 데이터 구축.
4. **교차 검증 및 업그레이드:** **Qwen-VL**에 [이미지 + 대사]를 프롬프트와 함께 입력하여, 음성 오타 교정 및 마크다운 형식의 완성도 높은 강의 노트 생성.

강의노트 자동 생성 및 교차 보정 AI 시스템
Multimodal Lecture Note Generation & Cross-Correction AI Architecture

1. 프로젝트 개요

본 시스템은 사용자가 업로드한 강의 영상으로부터 청각 정보(음성)와 시각 정보(화면 스크린샷 및 OCR)를 동시 추출하고, 이를 시간대별(Timestamp)로 매핑하여 상호 교차 검증 및 보정을 수행하는 혁신적인 스마트 강의노트 정리 AI 웹사이트입니다. 기존 STT 서비스의 한계인 전문 용어 오인식 및 뭉개진 발음 문제를 Vision-Language Model(Qwen)을 통한 교차 보정 파이프라인으로 해결

2. 전체 시스템 아키텍처 흐름
미디어 전처리 단계: 사용자가 영상을 웹사이트에 업로드하면, 백엔드 서버에서 1차적으로 오디오 파일(비압축 WAV/MP3)과 비디오 파일(MP4)로 분리합니다.
음성 인식 및 신뢰도 매핑 단계: 분리된 오디오를 Whisper AI에 전달하여 텍스트 데이터, 타임스탬프, 그리고 각 단어/문장별 확신도 점수(Confidence Score)를 추출합니다.
시각 정보 및 취약 구간 타겟 캡처 단계: 비디오 스트림을 분석하여 PPT 슬라이드가 전환되는 시점(Scene Change)을 감지해 캡처하고, 추가적으로 Whisper AI가 반환한 '확신도 점수 이하의 취약 구간 시간대'를 타겟팅 하여 정밀 화면 캡처를 실행합니다.
멀티모달 교차 검증 및 최종 보정 단계 (핵심): 서버 코드가 오역 위험이 있는 텍스트와 해당 시간대의 캡처본을 동기화하여 Qwen AI에게 동시 전달합니다. Qwen AI는 시각 이미지(PPT 내 전문 용어, 판서 등)와 텍스트 문맥을 교차 분석하여 최종 교정된 고품질 강의 노트를 생성합니다.

4. 핵심 교차 검증(Cross-Correction) 메커니즘

Qwen AI는 단순한 OCR 텍스트 추출기가 아니라 텍스트 추론 성능을 가진 멀티모달 LLM입니다. 따라서 별도의 타 LLM을 거치지 않고, Qwen 자체 프롬프트를 통해 추출과 수정을 한 번에 처리하는 효율적인 파이프라인을 구축합니다.

백엔드에서 Qwen AI에게 전달하는 프롬프트 엔지니어링 구조
서버는 캡처된 이미지 파일과 함께 아래와 같은 프롬프트 구조의 API 요청을 보냅니다.

[시스템 프롬프트 (System Prompt)]
너는 강의 노트 전문 교정 AI 편집자이다. 제공되는 [강의 화면 이미지]의 시각 정보(PPT 타이틀, 판서 텍스트,다이어그램 키워드)를 분석하고, 이를 [Whisper 오디오 텍스트]와 교차 검증하여 음성 인식 오류를 수정해야 한다.

[입력 데이터 (Input Data)]
- 타겟 시간대: 14분 20초
- Whisper 텍스트 (확신도 낮음): "이번 장에서는 디비전 모델의 수학적 원리를 자세하게 짚고 넘어갈게요."

[교정 규칙 (Rules)]
1. Whisper 텍스트 중 발음 오인식으로 판단되는 단어는 동일 시간대 이미지 속에 등장하는 정확한 전문 용어(특히 영문 키워드)를 참고하여 변경할 것.
2. 문장의 구조와 강사의 어조는 자연스럽게 유지하되, 핵심 키워드의 정확성을 최우선으로 확보
3. 교정 완료된 최종 강의 노트용 문장만 출력할 것.
4. 강의노트 생성 AI 웹사이트 구축 기획서 2

5. 기대 효과 및 서비스 차별점 (USP)

인간 검토의 최소화: AI가 자체적으로 시각과 청각 데이터를 대조하여 오타를 미리 1차 보정하므로, 사용자가 직접 수작업으로 수정해야 하는 리소스가 90% 이상 절감됩니다.
전문 용어 인식률 극대화: 음성만으로는 인식이 불가능한 고난도 전공 용어, IT 기술 스택, 의학 용어 등을 PPT 화면 데이터를 기반으로 정확하게 보정해 줍니다.
시각 자료 결합형 노트: 중요 화면 전환 시점의 캡처본과 정제된 텍스트가 시간대별로 예쁘게 레이아웃되어, 한 편의 완벽한 전자책 형태의 강의 노트를 자동으로 얻을 수 있습니다.
